    Tên đề tài 1 Xây dựng web thương mại điện tử tích hợp nhận diện sản phẩm bằng thị giác máy tính và phân tích dữ liệu
    Tóm tắt nội dung Đề tài tập trung xây dựng một hệ thống web thương mại điện tử tích hợp chức năng nhận diện sản phẩm dựa trên thị giác máy tính nhằm hỗ trợ người dùng tìm kiếm và đề xuất sản phẩm một cách trực quan, thông minh hơn. Bên cạnh đó, hệ thống thực hiện phân tích dữ liệu người dùng và giao dịch để thống kê, đánh giá hành vi mua sắm và hỗ trợ ra quyết định quản lý. Về công nghệ, đề tài đề xuất sử dụng Spring Boot để phát triển backend và các API xử lý nghiệp vụ, dữ liệu, cùng với React để xây dựng giao diện người dùng.
    Các yêu cầu cần đạt được:

    a. Xây dựng hệ thống web thương mại điện tử hoàn chỉnh

    b. Tích hợp chức năng nhận diện sản phẩm bằng thị giác máy tính, hỗ trợ người dùng tìm kiếm hoặc gợi ý sản phẩm thông qua hình ảnh.

    c. Thực hiện phân tích dữ liệu người dùng và giao dịch, cung cấp các thống kê, báo cáo phục vụ đánh giá hành vi mua sắm và hỗ trợ quản lý hệ thống.

    d. Thiết kế và triển khai hệ thống theo kiến trúc tách biệt frontend–backend, sử dụng Spring Boot cho API và React cho giao diện, đảm bảo khả năng mở rộng và dễ bảo trì.

    Tên đề tài 2 Thiết kế và triển khai hệ thống web thương mại điện tử C2C dựa trên kiến trúc Modular Monolith
    Tóm tắt nội dung Tóm tắt nội dung: Phát triển một hệ thống e-commerce sử dụng Modular Monolith, deploy như một ứng dụng duy nhất dựa trên Spring Boot. Tích hợp frontend web và ứng dụng di động để hiển thị sản phẩm. Sử dụng LLM (như GPT API) để tạo mô tả sản phẩm tự động, tạo chatbot hỗ trợ khách hàng, kết hợp phân tích dữ liệu (Scikit-learn hoặc TensorFlow) để gợi ý sản phẩm dựa trên lịch sử duyệt web.
    Các yêu cầu cần đạt được:

    a. Thiết kế và triển khai hệ thống e-commerce theo kiến trúc Modular Monolith, đảm bảo tính tách biệt module, dễ bảo trì và mở rộng.

    b. Xây dựng backend cung cấp API dùng chung cho frontend web và ứng dụng di động, hỗ trợ tất cả các chức năng của một ứng dụng thương mại điện tử.

    c. Tích hợp Large Language Models (LLM) như GPT API để tự động sinh mô tả sản phẩm và xây dựng chatbot hỗ trợ khách hàng trong quá trình tìm kiếm và mua sắm.

    d. Ứng dụng kỹ thuật phân tích dữ liệu và học máy để xây dựng hệ thống gợi ý sản phẩm dựa trên lịch sử duyệt web và hành vi người dùng.
